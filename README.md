情而已，心中震撼，已经不是他们所能形容的了。

送礼队伍旁若无人的从禁军中央穿过，路过乘影和御天身边，看也没看在原地怔忡的锦瑟，径直走向了城门口。

被沈尘提醒锦晏所在方位的天子近侍已经从城上下来，小跑着带领众人来到了锦晏所在的树下。

“奴才参见公主殿下！”

天子近侍一跪，押送金银珠宝的侍卫和在旁边观战的禁卫齐刷刷跪了下来，齐声高呼，“参见公主殿下！”

震天的声音将锦瑟从恐惧中拉回了现实。

她才说了句“平身”，便看到穿着一身绿色衣裙的锦晏从最高的树上飞了下来。

与她一
王福和禁军都愣了一下，他们没动手，可锦瑟的动作，确实是被人打耳光才有的躲闪。

好像有个谁也看不见的人站在锦瑟身边打她耳光一样。

他们又想起之前锦瑟那些古怪的举动，她说有鬼，她说自己被鬼缠上了……

原来不是锦瑟乱喊，是真的有鬼。

连着扇了十来个巴掌后，锦瑟的脑袋慢慢停止了摆动，但她的脸上却莫名出现了很多个层层叠叠的手掌印。

王福不动声色看了眼锦晏所在的方向，没看到锦晏有任何异常，却见锦晏身旁一红衣女子在同旁边的空气说着什么。

若真是有鬼，那能够看见鬼，还能够与鬼说话的女子，定然也不是寻常人。

而这些人，都站在公主殿下身后，一副以公主殿下为尊的样子。

饶是鬼又怎么样？

有些人，比鬼还可怕。

他不再多想，转头吩咐禁军，“此女一再对陛下和公主殿下出言不逊，将其打入死牢……”
